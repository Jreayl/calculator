# The Odin Project - Web Development #
## Project: Calculator ##
The final project in the 'Foundations' section of TOP is to make a web 
based calculator using HTML, CSS, and JavaScript. The current version is mostly
complete, but it is still a work in progress since there are still some bugs to iron out
and improvements to the Javascript code to be made.

### Features: ###
* power
* add
* subtract
* multiply
* divide
* negate
* all clear
* delete (backspace)
* decimals

### Features to be implemented: ###
* keyboard support
