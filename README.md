# The Odin Project - Web Development #
## Project: Calculator ##
The final project in the 'Foundations' section of TOP is to make a web 
based calculator using HTML, CSS, and JavaScript. This calculator has
keyboard support.

Keyboard shortcuts:
* Numbers (0-9)
* Operators (+, -, *, /, ^)
* Enter (=)
* Backspace (DEL)
* Escape (AC)
* ! (+/-)
